# Story 0020-001: Schedule Survey Run
**Estimate:** M

**Narrative:** As a team lead I want to schedule a survey run to open and close at specific times so that team members know when to participate.

**Acceptance Criteria:**
- Given I am a team lead, when I create or edit a survey run, then I can set a scheduled open time and close time.
- Given a survey run is scheduled, when the open time is reached, then the survey run opens automatically.
- Given a survey run is scheduled, when the close time is reached, then the survey run closes automatically.
- Given I view scheduled survey runs, when team members check, then they can see when the survey will be open.
- Given I try to schedule overlapping survey runs, when I submit, then I see a clear message that times cannot overlap.
