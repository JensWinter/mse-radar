# Story 0016-001: View Capability Profile
**Estimate:** M

**Narrative:** As a team member I want to see a visualization of my team's capability scores so that I can quickly understand our strengths and areas for improvement.

**Acceptance Criteria:**
- Given a survey run is closed with results, when I view the capability profile, then I see a visual representation of all capability scores.
- Given I view the capability profile, when I look at the visualization, then I can easily see the maturity level of each capability.
- Given I am a team member, when I access the results, then I see the latest survey run's capability profile by default.
- Given I am not a member of the team, when I try to view their capability profile, then I am denied access.
