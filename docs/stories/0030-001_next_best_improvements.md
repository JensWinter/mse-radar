# Story 0030-001: Next-Best Improvements
**Estimate:** M

**Narrative:** As a team member I want the system to suggest high-impact improvements so that we can prioritize our efforts effectively.

**Acceptance Criteria:**
- Given a survey run is closed with results, when I view improvement suggestions, then I see a prioritized list of recommended actions.
- Given the suggestions are based on our scores, when I review them, then they focus on capabilities where improvement would have the most impact.
- Given I view suggestions, when I read them, then each recommendation includes a clear rationale for why it was prioritized.
- Given our team has areas of disagreement, when I view suggestions, then the system considers confidence signals in its recommendations.
