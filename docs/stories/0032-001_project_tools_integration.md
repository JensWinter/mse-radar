# Story 0032-001: Project Tools Integration
**Estimate:** L

**Narrative:** As a team lead I want to connect improvement tasks to our project tools so that we can track improvement actions in our existing workflow.

**Acceptance Criteria:**
- Given I view improvement guidance, when I want to act on it, then I can create a linked task in our project backlog tool.
- Given I create a linked task, when I view the guidance later, then I can see that an action has been created.
- Given I want to connect to a project tool, when I configure the integration, then I can authenticate with supported external systems.
- Given I am not a team lead, when I try to configure integrations, then I am denied access.
