# Story 0026-001: Export Results
**Estimate:** M

**Narrative:** As a team member I want to export our survey results so that I can use them in workshops, reports, or other tools.

**Acceptance Criteria:**
- Given a survey run is closed with results, when I choose to export, then I can download the aggregated results.
- Given I export results, when I select a format, then I can choose from options like CSV, JSON, or PDF.
- Given I export results, when the file is generated, then it contains capability scores and summary information.
- Given I export results, when the file is generated, then it does not contain individual responses—only aggregated data.
- Given I am not a member of the team, when I try to export their results, then I am denied access.
