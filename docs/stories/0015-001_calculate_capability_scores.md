# Story 0015-001: Calculate Capability Scores
**Estimate:** M

**Narrative:** As a team member I want capability scores to be calculated from team responses so that we can understand our team's assessed level for each DORA capability.

**Acceptance Criteria:**
- Given a survey run is closed with responses, when I view the results, then I see a score for each DORA capability.
- Given multiple team members have responded, when scores are calculated, then each DORA capability score reflects the aggregated responses.
- Given a survey run is closed, when I view the results, then I see an overall summary of the team's DORA capabilities.
- Given scores are calculated, when I view them, then individual responses are not visible—only aggregated results.
