# Story 0021-001: Add Comments to Answers
**Estimate:** S

**Narrative:** As a team member I want to add comments to my survey answers so that I can provide context or explain my numeric selections.

**Acceptance Criteria:**
- Given I am answering a survey, when I respond to a capability question, then I can optionally add a free-text comment.
- Given I am answering a survey, when I add a comment to my answer, then the comment is saved immediately.
- Given I view my own response, when I check my answers, then I can see any comments I added.
- Given I view aggregated team results, when I look at capability scores, then individual comments are not visible to protect privacy.
