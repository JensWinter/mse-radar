# Story 0022-001: Question Versioning
**Estimate:** M

**Narrative:** As a team member I want survey questions to be versioned so that historical survey runs remain interpretable even when questions change.

**Acceptance Criteria:**
- Given the survey questions have been updated, when I view an older survey run, then I see the questions as they were when that survey was conducted.
- Given the survey model has multiple versions, when a new survey run is created, then it uses the current version of the questions.
- Given I compare results across survey runs, when the survey versions differ, then I am made aware of the version difference.
- Given questions are versioned, when I view historical results, then the scores remain meaningful in the context of the original questions.
