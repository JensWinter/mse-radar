# Story 0033-001: Multi-Language UI
**Estimate:** L

**Narrative:** As a user I want to use the application in my preferred language so that I can better understand the interface and content.

**Acceptance Criteria:**
- Given I access the application, when I select a language preference, then the interface displays in my chosen language.
- Given I change my language preference, when I navigate the application, then all UI elements appear in the selected language.
- Given guidance content is available in multiple languages, when I view it, then it displays in my preferred language.
- Given my preferred language is not available, when I use the application, then I see content in a default language with a clear indication.
