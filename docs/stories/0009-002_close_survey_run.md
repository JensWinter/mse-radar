# Story 0009-002: Close Survey Run
**Estimate:** M

**Narrative:** As a team lead, I want to close a survey run so that a response collection ends and results can be analyzed.

**Acceptance Criteria:**
- Given I am a team lead with an open survey run, when I close it, then the survey run stops accepting responses.
- Given a survey run is closed, when team members try to submit responses, then they see a clear message that the survey is closed.
- Given a survey run is closed, when team members view results, then they can see the aggregated capability scores.
- Given I am not a team lead, when I try to close a survey run, then I cannot close it.
