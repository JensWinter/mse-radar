# Story 0003-001: Create a Team
**Estimate:** M

**Narrative:** As an authenticated user, I want to create a new team so that I can assess and improve my team's software engineering capabilities.

**Acceptance Criteria:**
- Given I am signed in, when I create a team with a name and description, then the team is created and I become the team lead.
- Given I have created a team, when I view my teams, then the new team appears in my list of teams.
- Given I have created a team, when I view its details, then I see the team name, description, creation date, and myself as team lead.
