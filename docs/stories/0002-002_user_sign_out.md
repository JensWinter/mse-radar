# Story 0002-002: User Sign Out
**Estimate:** S

**Narrative:** As a signed-in user, I want to sign out of the application so that I can protect my account when I'm done using it.

**Acceptance Criteria:**
- Given I am signed in, when I choose to sign out, then my session ends, and I am returned to a public page.
- Given I am signed in, when I sign out and when I try to access a protected page, then I am redirected to the sign-in page.
