# Story 0025-001: Confidence Indicators and Disagreement Visualization
**Estimate:** M

**Narrative:** As a team member I want to see where team responses vary significantly so that we can identify areas requiring discussion and alignment.

**Acceptance Criteria:**
- Given a survey run is closed with results, when I view a capability score, then I see an indicator of how much agreement or disagreement there was among responses.
- Given there is high variance in responses for a capability, when I view the results, then that capability is highlighted as an area of disagreement.
- Given there is strong agreement among responses, when I view the results, then I see a high confidence indicator for that capability.
- Given I view disagreement indicators, when I identify misalignment areas, then I can use this information to guide team discussions.
