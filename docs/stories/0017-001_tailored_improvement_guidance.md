# Story 0017-001: Tailored Improvement Guidance
**Estimate:** L

**Narrative:** As a team member I want to receive improvement guidance tailored to our assessed capability levels so that we know how to improve in areas where we are weak.

**Acceptance Criteria:**
- Given a survey run is closed with results, when I view a capability from survey run results, then I see improvement guidance for that specific capability.
- Given I view improvement guidance for a capability, when I read it, then it is appropriate to our current capability level.
- Given I view improvement guidance for a capability, when I read it, then it gives actions to take and shows its DORA source.
