# Story 0011-002: Edit Survey Response
**Estimate:** S

**Narrative:** As a team member, I want to update my answers while the survey is open so that I can correct my selections before the survey closes.

**Acceptance Criteria:**
- Given I have submitted a response and the survey run is still open, when I select a different value for a question, then my updated answer replaces the previous one immediately.
- Given I edit my response multiple times, when the survey closes, then only my most recent answers are used for results.
- Given the survey run is closed, when I try to access the survey, then I see a message that it cannot be edited.
- Given I edit my response, when I view my submission afterward, then I see my updated answers.
