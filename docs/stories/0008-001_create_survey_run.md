# Story 0008-001: Create Survey Run
**Estimate:** M

**Narrative:** As a team lead, I want to create a new survey run for my team so that we can assess our capabilities at different points in time.

**Acceptance Criteria:**
- Given I am a team lead, when I create a survey run, then a new assessment cycle is created for my team.
- Given a survey run is created, when team members view the team, then they see the new survey run in the list.
- Given I am not a team lead, when I try to create a survey run, then I am denied access.
- Given I create a survey run, when I view its details, then I see it is in a pending state.
