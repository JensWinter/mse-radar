# Story 0001-001: User Registration
**Estimate:** M

**Narrative:** As a new user, I want to create an account so that I can access the application and participate in team surveys.

**Acceptance Criteria:**
- Given I am on the registration page, when I provide a valid email address and password, then my account is created and I receive confirmation.
- Given I try to register with an email already in use, when I submit the registration form, then I see a clear message that the email is already registered.
- Given I provide a password that does not meet security requirements, when I submit the form, then I see a clear message explaining what is required.
- Given my account is successfully created, when I return to the application, then I can sign in with my credentials.
