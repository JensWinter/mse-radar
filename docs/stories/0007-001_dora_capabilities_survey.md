# Story 0007-001: DORA Capabilities-Based Survey
**Estimate:** L

**Narrative:** As a team member I want to answer a survey based on DORA capabilities so that my team can assess our software engineering skills using an industry-recognized framework.

**Acceptance Criteria:**
- Given I access a survey, when I view the questions, then each question corresponds to one DORA capability.
- Given I view the survey, when I read a question, then I understand what capability is being assessed.
- Given the survey is based on DORA capabilities, when I complete all questions, then I have assessed the full set of capabilities defined by the survey model.
