# Story 0031-001: Chat Assistant
**Estimate:** L

**Narrative:** As a team member I want to ask questions about capabilities through a chat interface so that I can get quick answers and find relevant guidance.

**Acceptance Criteria:**
- Given I have a question about a capability, when I use the chat assistant, then I receive a helpful response about that capability.
- Given I ask about improvement strategies, when I use the chat assistant, then it provides guidance relevant to my question.
- Given I receive a response from the chat assistant, when I want more details, then it can link me to relevant guidance content in the application.
- Given I use the chat assistant, when I ask questions, then the responses are grounded in DORA research and the application's content.
